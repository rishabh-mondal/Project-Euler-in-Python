2. Mutable vs. Immutable Data Types: Explain the difference between mutable and immutable data types in Python. Provide examples of each.¶

Mutable Data Types: A mutable object is one whose state or value can be changed after it is created. You can modify, add, or remove elements from mutable objects without creating a new object.

Immutable Data Types: An immutable object is one whose state or value cannot be changed after it is created. Once created, you cannot modify its contents, but you can create a new object if you need to change the value.

Difference¶

Mutable Data Types:

Modification: You can directly change the contents of these objects. For example, you can append, remove, or modify elements in a list or dictionary. Performance: Modifying mutable objects is generally faster, especially when dealing with large datasets, because no new object needs to be created when the contents change. Memory: Mutable objects may have shared references, meaning multiple variables can point to the same object in memory, leading to potential side effects if one is modified. Examples of Mutable Data Types: Lists, dictionaries, and sets are all mutable in Python.

Immutable Data Types:

Modification: You cannot modify the content of these objects directly. If you want to change the value, a new object must be created. Performance: Modifying immutable objects often involves creating a new object, which may be less efficient in some cases compared to mutable objects. Memory: Immutable objects generally use less memory, as they cannot be modified, and the same object can be safely shared across different parts of a program. Examples of Immutable Data Types: Integers, strings, tuples, and frozensets are immutable in Python.

def fibo(n):
    fibo_list=[0,1]
    for i in range(2,n):
        next_fibo=fibo_list[i-1]+fibo_list[i-2]
        fibo_list.append(next_fibo)
    return fibo_list[:n]

def character_frequency(s):
    s = s.lower()
    freq_dict = {}
    for char in s:
        if char.isalnum():  
            freq_dict[char] = freq_dict.get(char, 0) + 1
    
    return freq_dict

## Write a function to sort a list of dictionaries by a key.
def sort_list_dists(data, key):
    for i in range(len(data)):
        for j in range(0,len(data)-i-1):
            if data[j][key]>data[j+1][key]:
                data[j],data[j+1]=data[j+1],data[j]
    return data

data = [{'name': 'Alice', 'age': 25}, {'name': 'Bob', 'age': 22}]
sorted_data = sort_list_dists(data, 'age')

## Write a Python function to normalize a list of numbers to a range of 0 to 1.
def normalize(numbers):
    min_val = min(numbers)
    max_val = max(numbers)
    return [(x - min_val) / (max_val - min_val) for x in numbers]

numbers = [10, 20, 30, 40, 50]
normalized_numbers = normalize(numbers)
print("Normalized Numbers:", normalized_numbers)
